## 🌌 The Swarm Philosophy
Each repository under the CozanetOS organization houses a singular, ultra-optimized micro-engine. Dynamic, string-addressed messaging allows our components to act as unified neural nodes without tight architectural coupling.

---

## 📂 Core Ecosystem Repositories

- **[cozanet-os](https://github.com/CozanetOS/cozanet-os)**: The root monorepo containing bootstrapping utilities and workspace guides.
- **[cozanet-core](https://github.com/CozanetOS/cozanet-core)**: Low-level system scheduling and execution pipelines.
- **[cozanet-ceo](https://github.com/CozanetOS/cozanet-ceo)**: The central orchestration intelligence engine.
- **[cozanet-comms](https://github.com/CozanetOS/cozanet-comms)**: The main Pub-Sub message broker.
- **[cozanet-groq](https://github.com/CozanetOS/cozanet-groq)**: Fast, dynamic LLM layer with multi-key rate-limit rotation.
- **[cozanet-security](https://github.com/CozanetOS/cozanet-security)**: Security policies, key validation, and process sandboxing.
