# Security Policy

CozanetOS takes security extremely seriously. Because our AI agents execute custom actions, access toolboxes, and communicate dynamically, strict enforcement of sandbox isolation and process validation is critical.

---

## Supported Versions

We actively support security updates for the following versions:

| Version | Supported |
| --- | --- |
| 1.0.x |  Active |
| < 1.0 |  End of Life |

---

## Reporting a Vulnerability

Please do not report security vulnerabilities publicly via GitHub Issues. Instead, follow the process below:

1. **Email us**: Send a detailed security report to `security@cozanet.io`.
2. **Details**: Include a proof of concept, the affected engine ID, and potential risk mitigation strategies.
3. **Response**: We will acknowledge receipt of your report within 24 hours and keep you updated on remediation progress.
